hexo创建新文章并提交

Sep 01, 2019点击

1. 创建一个新文章

hexo new '文章的title' 或者 hexo new post '文章的title'

2. 写文章

注意：提交文件的文件格式都是 .md 文档，本人用的是 Typora 写的md文档

3.提交文章

1. `hexo g`   --  生成静态文件
2. `hexo s`   --  启动本地服务，查看效果（如果确定可以省略）
3. `hexo d`   --  上传到服务器
4. 可以在线上查看提交的博客文章了
